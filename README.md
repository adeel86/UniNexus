# UniNexus

UniNexus is a Gen Z student ecosystem platform that connects students, teachers, universities, and industry partners. The platform features AI-powered learning assistance, gamification, social networking, and career development tools.

## What is UniNexus?

UniNexus is designed to be the ultimate learning and career development hub for students. It brings together:

- **Students** - Connect with peers, track learning progress, earn badges, and get AI-powered tutoring
- **Teachers** - Upload course materials, endorse students, and monitor student engagement
- **University Administrators** - View retention metrics and institutional insights
- **Industry Partners** - Discover talent, post challenges, and provide feedback

## Key Features

### For Students
- **Social Feed** - Share posts, react, and engage with peers
- **AI CareerBot** - Get personalized tutoring based on course materials
- **Gamification** - Earn badges, track streaks, and climb leaderboards
- **Course Enrollment** - Browse and enroll in courses
- **Challenges** - Participate in coding challenges and hackathons
- **Groups** - Join study groups and communities

### For Teachers
- **Course Management** - Create and manage courses
- **Content Upload** - Upload lecture notes, slides, and materials for AI integration
- **Student Analytics** - View student engagement and progress
- **Endorsements** - Endorse student skills

### For University Admins
- **Dashboard** - View retention metrics and institutional insights
- **Course Validation** - Approve or reject course submissions
- **Announcements** - Post university-wide announcements

### For Industry Partners
- **Talent Discovery** - Search for skilled students
- **Challenges** - Create and sponsor coding challenges
- **Feedback** - Provide interview feedback to students

## Technology Stack

- **Frontend**: React with TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Neon)
- **Authentication**: Firebase Auth (with development bypass for demo accounts)
- **AI**: OpenAI integration for intelligent tutoring
